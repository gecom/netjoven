Pasos para migración 

1) Ejecutar  el primer archivo estructura_netjoven.sql
2) una vez terminado el primer paso ejecutamos este archivo store_procedure.sql
3) Terminado el segundo paso ejecutamos el siguiente archivo netjoven_script_1.sql, este script es que se encargara de migrar todos los datos a las nuevas tablas.
4) Una vez terminado el paso anterior modificar el siguiente store procedure add_post_search, para que ya no pueda  pasarle ningún parámetro y agregarle la condición de que pueda obtener los ultimo 15 notas que hayan sido modificadas.  Después crear un job por mysql del store procedute add_post_search  para que se ejecute cada 15 minutos.
